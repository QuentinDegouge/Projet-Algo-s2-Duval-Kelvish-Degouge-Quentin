Le projet CDataframe a pour mission de créer un CDataframe (librairie) capable de faciliter la manipulation des données,
c'est à dire un ensemnble de colonnes portant chacune un titre et permetant de stocker un nombre indéfini de données de
même type.

lorsque le programme se lance, il est demandé dans l'ordre à l'utilisateur de :

1 - Agir sur l'alimentation
	-choisir la taille du dataframe
	-nommer la première colonne
	-choisir le nombre de valeurs de la première colonne
	-rentrer les valeurs dans cette dernière
	-nommer la seconde colonne
	-choisir le nombre de valeurs de la seconde colonne
	-rentrer les valeurs dans cette dernière

2 - afficher le dataframe
	-choisir jusqu'à quelle ligne afficher les nombres
	-choisir jusqu'à quelle colonne afficher les nombres

3 - effectuer des opération usuelles
	-choisir une valeur à ajouter pour chaque colonne
